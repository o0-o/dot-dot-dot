# dot-dot-dot
My dot files

# Requires
* tmux
* tmuxinator
* neovim
* zsh

# Installation
Install into your home directory
```
cd &&
git init &&
git remote add origin https://github.com/o0-o/dot-dot-dot.git &&
git fetch origin &&
git checkout origin/master --track &&
git submodule update --init --recursive
```
